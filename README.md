# Site24x7 Demo Apps

This repository contains demo applications and automated deployment scripts for showcasing **ManageEngine Site24x7's full-stack observability** capabilities — including APM, infrastructure monitoring, log management, and real user monitoring across polyglot microservice architectures.

---

## Applications

| Application | Description | Tech Stack |
|-------------|-------------|------------|
| [ZylkerKart E-Commerce](Applications/ZylkerKart_Ecommerce/) | Polyglot microservices e-commerce platform with 6 services across 6 languages | Java, Node.js, Go, Python, C#/.NET, MySQL, Redis |

## Terraform Deployment

The `Terraform Deployment/` folder contains automated Terraform scripts for provisioning infrastructure and deploying the demo applications.
